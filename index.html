Sevenish

This is the first font I’ve created in a long time. A lot has changed since 1989 when I created bitmap fonts with ResEdit. I need to catch up.

Status

This font matches the picture in my head! This release covers all of the necessities.

Bold is a work in progress. I have created all of the glyphs that only use horizontal and vertical lines. This style is working so I will continue.

Click here for a look at the prototype for this font.

Purpose

Curiosity

I use fonts all the time in my web pages. I’m always trying out new features. I want to know what this tech looks like to the implementor.

Fun

I can poke around at this stuff a little at a time. It’s like doodling.

Tools

I created this font using Birdfont.

Style

This font was inspired — but not limited — by a 7 segment display. The digits are pretty standard. The other characters have more freedom, but they are all clearly part of a set.

The basic shape is a hexagon. The sides adjacent to a right angle are all the same size. The remaining two sides can grow or shrink as required. I looked at some other options for the ends, but this looks the best at a number of different sizes.

I still don't have the diagonal lines down perfect. I can't get the length to be exactly what I need. With horizontal or vertical lines I can just move my mouse until I get to the right place.

I find bright letters on a dark background make this font the most legible. With black on white consider using the CSS letter-spacing property.
