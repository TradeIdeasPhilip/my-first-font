My First Font

This is the first font I’ve created in a long time. A lot has changed since I created bitmap fonts with ResEdit. I need to catch up.

Status

I’ve created a “Minimum Viable Product.”

Clearly there’s a lot of work to do to make this usable. The most obvious problem is the kerning; the spacing between the letters is inconsistent enough to make the font hard to read.

Purpose

Curiosity. I use fonts all the time in my web pages. I’m always trying out new features. I want to know what this tech looks like to the implementor.

Fun. I can poke around at this stuff a little at a time. It’s like doodling.

Tools

I created this font using Birdfont.

Style

This font was inspired by a standard 7 segment display. The digits are pretty standard. The other characters have more freedom, but they keep the basic style.

The basic shape is a circle which I stretched to different lengths. I tried to keep the widths the same, but I had trouble with the diagonal lines.

I would like to replace the basic shape with something more like a line segment with round end caps. The end caps would not change regardless of the line’s length. The end caps might be right triangles, so four of them would fit together tightly.
