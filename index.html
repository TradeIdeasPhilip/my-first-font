Sevenish

This is the first font I’ve created in a long time. A lot has changed since 1989 when I created bitmap fonts with ResEdit. I need to catch up.

Status

This font matches the picture in my head! This release covers all of the necessities.

Bold is a work in progress. I have created most of the glyphs. I had to change some of the regular glyphs to make room for the bold glyphs. This style is working so I will continue.

Click here for a look at the prototype for this font.

Purpose

Curiosity

I use fonts all the time in my web pages. I’m always trying out new features. I want to know what this tech looks like to the implementor.

Fun

I can poke around at this stuff a little at a time. It’s like doodling.

Tools

I created this font using Birdfont.

Style

This font was inspired — but not limited — by a 7 segment display. The digits are pretty standard. The other characters have more freedom, but they are all clearly part of a set.

The basic shape is a hexagon. The sides adjacent to a right angle are all the same size. The remaining two sides can grow or shrink as required. I looked at some other options for the ends, but this looks the best at a number of different sizes.

I still don't have the diagonal lines down perfect. I can't get the length to be exactly what I need. With horizontal or vertical lines I can just move my mouse until I get to the right place.

I find bright letters on a dark background make this font the most legible. With black on white consider using the CSS letter-spacing property.

Italic

I looked into an italic version of this font. Here's something I created with diagonal lines.

That looks good, but I get a similar effect for free.

Next I tried some curves. I kept the pointy ends exactly the same. Those are 45-45-90 triangles. The short sides are 3 grid squares long and all 45° from a cardinal direction. But the tops are now offset from the bottoms. The idea was to use Bézier curves to connect to tops and bottoms, instead of straight line segments. The idea was to make the top and bottom of each vertical segment straight, just like the regular version. Then to curve in the middle to make the top and bottom meet.

Some versions looked amusing when they were huge. But as they got smaller, they just looked like imperfect hexagons.

These two approaches were both dead ends. For now I can use the automatic/free italics. But I would like to revisit my first prototype. Those ovals gave the font a very round feeling, perfect for an italic font. I'd lay the ovals out in the same basic shape as the hexagons, above. I.e. horizontal lines slide and vertical lines tilt.
